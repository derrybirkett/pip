# Security Policies

## Baseline Controls
- Least privilege access; MFA required
- Secrets management: vault or managed secrets
- Encryption: in transit (TLS) and at rest

## SDLC
- Security review in design; threat modeling
- Dependency scanning and SAST in CI
- Code review with security checklist

## Data Handling
- Data classification and retention
- Backups and recovery objectives

## Incident Response
- Severity levels, roles, and runbooks
- Post-incident review and action tracking
