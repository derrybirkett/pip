# Product Ops Playbook

## Capture Opportunities
- Intake channels: support, sales, analytics, research
- Triage: score by impact, effort, confidence

## Plan Features
- Map to goals and metrics
- Write one-pagers; define scope and DoD

## Run Discovery
- Problem/solution interviews; prototype tests
- Record insights and decisions

## Handoff to Delivery
- Align on data models, API contracts, and flows
- Create tickets with clear acceptance criteria

## Measure & Iterate
- Instrument events; monitor activation and retention
- Close the loop with changelog and blog release
